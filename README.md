# Student Management System

A web-based Student Management System developed using PHP, MySQL, JavaScript, AJAX, HTML, and CSS. It provides secure user authentication, OTP-based password reset, and student record management.

---

## Features

- User Registration
- User Login
- Forgot Password (OTP Verification)
- Reset Password
- Session Authentication
- Student CRUD (Create, Read, Update, Delete)
- Profile Image Upload
- Responsive Design
- Password Hashing
- Secure Prepared Statements

---

## How to Run

1. Install XAMPP.
2. Copy the project into the `htdocs` folder.
3. Start Apache and MySQL.
4. Import the database into phpMyAdmin.
5. Open the project in your browser.
